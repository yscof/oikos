요약

오이코스는 어떠한 개인정보도 수집하지 않습니다.

데이터 저장

앱에 입력한 모든 기록(금액, 카테고리, 메모, 날짜)은 사용자의 기기 안에만 저장됩니다. 서버로 전송되지 않으며, 개발자를 포함한 누구도 접근할 수 없습니다. 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.

수집하지 않는 것

  • 계정·로그인 없음 — 이름, 이메일, 전화번호를 요구하지 않습니다.
  • 네트워크 전송 없음 — 분석 도구, 광고 SDK, 추적 기술을 사용하지 않습니다.
  • 기기 권한 최소화 — 카메라, 위치, 연락처 등 민감한 권한을 요청하지 않습니다.

데이터 내보내기

설정 화면의 "데이터 내보내기"는 사용자가 직접 실행할 때만 기록을 JSON 텍스트로 기기 클립보드에 복사합니다. 이 과정에서도 데이터는 기기를 벗어나지 않습니다.
